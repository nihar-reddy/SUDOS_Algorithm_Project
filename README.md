# 🚚 SUDOS – Smart Urban Delivery Optimization System

## 📌 Overview
SUDOS is a project developed for the Design and Analysis of Algorithms (DAA) course.  
It models a city as a **graph**, where:
- Nodes = locations (warehouse + customers)
- Edges = roads (with distances)

The system applies multiple algorithms to optimize delivery routes and agent assignments.

---

## 🎯 Objective
- Minimize total delivery distance and time  
- Assign delivery agents efficiently  
- Compare different DAA algorithms  

---

## ⚙️ Algorithms Used

- **Greedy (Nearest Neighbor)** – Fast but not always optimal  
- **Dijkstra Algorithm** – Shortest path from warehouse  
- **Floyd-Warshall Algorithm** – All-pairs shortest paths  
- **Dynamic Programming (TSP)** – Optimal for small graphs  
- **Approximation TSP** – Faster, near-optimal  

---

## 🖥️ Features
- Add locations and routes  
- Set warehouse and customers  
- Multi-agent simulation  
- Algorithm selection  
- Visual route execution  
## 📊 Key Insight

- Greedy → fastest  
- Dijkstra → optimal shortest paths  
- DP-TSP → fully optimal but slow  
- Approximation → scalable  

👉 No single algorithm is best for all cases.

---

## 🌍 Applications
- E-commerce delivery (Amazon, Flipkart)  
- Food delivery (Swiggy, Zomato)  
- Ride-sharing (Uber, Ola)  
- Logistics optimization
